# HBNix

A Nix library for packaging and building various console homebrew software using their respective SDKs.

## Supported Platforms

* PSVita using [vitasdk](https://vitasdk.org/) and [vitasdk.nix](https://github.com/sleirsgoevy/vitasdk.nix)
* Nintendo 3DS (planned)
